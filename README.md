# Django-app

## About
To-do list app implemented in django framework.

## Get started:

Download using following command in terminal:

`git clone https://github.com/crimsonKn1ght/django-app.git`

Now go to directory using:

`cd django-app`

Create environment if you wish using:

`python -m venv env_name`

Activate the `env`:

Windows:

`env_name\Scripts\activate.bat` or `env_name\Scripts\Activate.ps1`

Linux:

`source env_name/Scripts/activate`

## Install requirements:

Use `requirements` file to install dependencies:

`pip install -r requirements.txt`

## Migrate and start server:

Migrate any changes using:

`cd app/ && python manage.py makemigrations`

`python manage.py migrate`

Now, lets start the server:

`python manage.py runserver`

Go to [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
